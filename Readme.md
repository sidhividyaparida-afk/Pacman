## 📖 About the Project

This project is a **Pacman-inspired arcade game** developed using **Python** and the **Turtle Graphics** module.

The player controls Pacman through a maze, collecting food while avoiding enemies. It demonstrates core programming concepts like:

- Object-Oriented Programming (OOP)
- Collision Detection
- Event Handling
- Game Loop
- Keyboard Controls
- Turtle Graphics Animation

This project is perfect for beginners learning game development with Python.
